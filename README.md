# Cuaderno de Obsidian – Desarrollo de Software

## Descripción  
Este cuaderno agrupa notas, ejercicios y recursos académicos del semestre de *Desarrollo de Software*. Funciona como una bóveda centralizada donde encuentras la teoría, ejemplos de código y materiales de estudio organizados para facilitar el repaso y la referencia rápida.

## Uso de Obsidian  
1. **Clona el repositorio** en tu máquina local.  
2. **Abre la carpeta** raíz del proyecto con Obsidian (Archivo → Abrir carpeta como vault).  
3. **Activa los plugins recomendados** (por ejemplo, *Kanban*, *Daily notes*, *Obsidian Git* si deseas sincronizar cambios).  
4. Navega por las distintas secciones mediante los enlaces internos (`[[...]]`).  
5. (Opcional) Conecta con **NotebookLM** o cualquier otro motor de IA para consultas avanzadas usando los enlaces proporcionados en la sección *Recursos y AI*.

## Contenido académico  
- **01_Clases/** – Apuntes diarios estructurados por semana, con ejemplos de código y diagramas.  
- **02_Recursos/Lecturas/** – PDFs y libros de referencia (Larman, Robert C. Martin, Fowler, etc.).  
- **04_AI_Assets/** – Materiales y consultas realizadas a herramientas de IA.  
- **05_Evaluaciones/** – Enlaces a los parciales y al proyecto final, con notas de estudio y resultados.  
- **Plan de Estudios** – Cronograma del semestre, entregas y evaluaciones.

## Audiencia  
Diseñado para **estudiantes** que buscan repasar los conceptos clave del curso y consolidar su aprendizaje mediante notas interactivas y recursos centralizados. No incluye licencia; el repositorio está pensado para uso interno y colaborativo entre compañeros de clase.
